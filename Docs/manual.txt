Справка

Граф можно ввести с клавиатуры, при помощи мыши, считать из файла.

Чтобы считать граф из файла нужно нажать кнопку "From File" -> выбрать файл, который содержит граф -> нажать на кнопку "Открыть файл".

Ввести граф с клавиатуры или при помощи мыши вам помогут следующие команды:

Добавить вершину:

1) Ввести в поле "Name" имя вершины -> нажать кнопку "add Node".

2) Ввести в поле "Name" имя вершины -> нажать ЛКМ на рабочее поле.

Добавить ребро:

1)Ввести в поля "From" и "To" названия вершин -> ввести в поле "Edge Weight" вес ребра -> нажать кнопку "add Edge" (можно вводить вершины, которые еще не созданы, они появятся автоматически).

2)Выбрать вершину, из которой ребро будет выходить -> нажать на нее ПКМ (вершина должна покрасится в синий цвет) -> выбрать вершину, в которую ребро будет входить -> нажать на нее ПКМ 2 раза (вершина должна покрасится в желтый цвет) -> ввести в поле "Edge Weight" вес ребра -> нажать клавишу "add Edge".

Вы можете изобразить граф в различных координатах, для этого нужно нажать клавишу "Repaint Graph".

Можете сохранить свой граф в файл нажав кнопку "Save this Graph".

Вы можете самостоятельно выбирать координаты для вершин, для этого вам нужно ввести в поле "Name" имя вершины, которую вы хотите переместить -> нажать ЛКМ на рабочем поле на место, куда вы хотите поставить вершину.

Когда граф считан или построен, переходим к просмотру алгоритма, следует нажать на кнопку "go to algorithm".

Для того, чтобы запустить алгоритм нужно назначить стартовую вершину, для этого следует ввести в поле "input start Node" имя вершины -> нажать клавишу "start algorithm".

После запуска алгоритма вы можете узнать расстояние от стартовой вершины к заданной. Для этого нужно ввести в поле "A im Node" имя заданной вершины -> нажать клавишу "Get lenght". Ответ появится в сообщении.

Просмотреть работу алгоритма можно нажав кнопку "start/end" и пошагово, нажимая клавиши "next step" и "previous step".

Чтобы начать работу с новым графом нужно нажать кнопку "Reset".

Чтобы прекратить работу нужно нажать на крестик в правом верхнем углу приложения.
